# BD-Project

Проект выполнил: Мальцев Сергей

База данных для сервиса доставки продуктов

### Таблицы:
- *users* - Пользователи
- *shopping_carts* - Корзины товаров
- *cart_goods* - Товары внутри корзин
- *goods* - Сами товары
- *orders* - Заказы
- *delivery_agents* - Доставщики
